# ZDS Component Index (L0)

> **用途**：锚定层常驻，Phase 1-4 全程可见。让 Router 和所有 Skill 在任意阶段都能感知 ZDS 有哪些组件可用。
> **维护**：ZDS 新增/废弃组件时由设计师更新。

| ID | 组件名 | 一句话描述 | 适用场景 |
|----|--------|-----------|---------|
| zds-button | Button | 主操作按钮，支持 primary/secondary/ghost/danger 变体 | 表单提交、CTA、操作触发 |
| zds-card | Card | 内容容器卡片，白底圆角阴影 | 仪表盘、信息展示、列表项 |
| zds-table | Table | 数据表格，含排序和行悬停 | 数据列表、管理面板 |
| zds-input | Input | 文本输入框，含 focus 态和校验态 | 表单、搜索、筛选 |
| zds-modal | Modal | 模态弹窗，含遮罩层 | 确认操作、表单填写、详情预览 |
| zds-sidebar | Sidebar | 侧边导航栏，支持折叠 | 应用主导航 |
| zds-badge | Badge | 标签/徽章 | 状态标记、分类、计数 |
| zds-avatar | Avatar | 用户头像，圆形 | 用户标识、参与者列表 |
| zds-empty-state | Empty State | 空状态占位，含插画+引导文案 | 列表/表格无数据时 |
| zds-tabs | Tabs | 选项卡切换 | 内容分类、视图切换 |
| zds-dropdown | Dropdown | 下拉菜单 | 操作菜单、筛选选项 |
| zds-tooltip | Tooltip | 悬浮提示 | 信息补充、图标说明 |
| zds-toast | Toast | 轻量级通知提示 | 操作反馈、状态变更提示 |
| zds-breadcrumb | Breadcrumb | 面包屑导航 | 层级页面导航 |
| zds-pagination | Pagination | 分页控件 | 列表/表格分页 |
| zds-toggle | Toggle | 开关切换 | 设置项启用/禁用 |
| zds-checkbox | Checkbox | 复选框 | 多选、同意确认 |
| zds-radio | Radio | 单选按钮 | 互斥选项选择 |
| zds-progress | Progress | 进度条 | 上传进度、步骤进度 |
| zds-skeleton | Skeleton | 骨架屏加载占位 | 内容加载中 |
